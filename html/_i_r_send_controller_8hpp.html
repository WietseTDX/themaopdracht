messageBitPrinter

This function prints a parameter in bits. The function is templated. The first template parameter is the amount bit in the variable type. The second template variable is the type. This parameter is not obligated. The function it self expects a variable of a type. Call example: bitPrinter<16>( (uint16_t) message);
